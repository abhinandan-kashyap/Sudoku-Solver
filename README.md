# 🧩 Sudoku Solver

## 📌 Project Overview

Sudoku Solver is a **Java-based algorithm project** that solves Sudoku puzzles using the **Backtracking Algorithm**.

The project includes both a console-based implementation and an interactive web interface, demonstrating problem-solving, recursion, and algorithm design skills.

---

## 🚀 Features

- Automatic Sudoku solving
- Backtracking algorithm implementation
- Console & UI-based versions
- Sudoku board validation
- Fast and accurate solution generation
